## Naming conventions

A components name must start with an uppercase character

### General naming conventions

- Use PascalCase for namespaces, types, and members.
- Use camelCase for parameters and local variables.
- Use a verb-object pair for method names.
- Use a noun for property names.
- Use a noun or noun phrase for enum types.
- Use a singular noun for enum values.
- Use a noun for event names.
- Use a noun or noun phrase for constant values.
