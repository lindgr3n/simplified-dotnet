# IAsyncEnumerable

`IAsyncEnumerable` interface is used to provide a way to iterate over a collection asynchronously. It is a part of the `System.Collections.Generic` namespace. ## Example

```csharp
using System;
using System.Collections.Generic;
using System.Threading.Tasks;

class Program
{
    static async Task Main()
    {
        await foreach (var number in GenerateSequence())
        {
            Console.WriteLine(number);
        }
    }

    static async IAsyncEnumerable<int> GenerateSequence()
    {
        for (int i = 0; i < 20; i++)
        {
            await Task.Delay(100);
            yield return i;
        }
    }
}
```

In the above example, we have created a method `GenerateSequence` that returns an `IAsyncEnumerable<int>`. We are using the `await foreach` loop to iterate over the collection asynchronously.
